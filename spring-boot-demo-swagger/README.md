## Spring Boot 集成 Swagger 接口文档框架演示

### 1️⃣使用 knife4j 实现

因 `Swagger` 默认的 `UI` 实现不是很好，所以建议使用 `knife4j`。 

knife4j 是为 Java MVC 框架集成 Swagger 生成 Api 文档的增强解决方案。

🌍官方文档：https://doc.xiaominfo.com/knife4j/documentation/get_start.html
